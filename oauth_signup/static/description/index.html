Signup With Social Media Provider

Easy without configuration

Configuration

Configure your favourite provider in a second

Configure with Google, Facebook, Twitter or LinkedIn and provide easy way to signup user in odoo web or website.

Set Provider Credential

Go to your service provider and get client_id and client_secret. If you need any help to get credential, please refer help link in page which help you configure your provider. There is no need to configure any other parameter rather than client_id and client_secret all other are set by default.

User Signup

App will fetch required info from provider and set it in odoo user

While user signup with provider app will fetch user name, email and image and set it in newly created user.

If you need any help or want to add more social providers don't hesitate to contact odoosoft or mail us on info@odoosoft.com
In order to work correctly App need to install requests-oauthlib python dependacy on your server
